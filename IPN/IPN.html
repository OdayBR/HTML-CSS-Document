1:The Basics of Web and HTML

How the Web Works

The Web is a system of communication between your computer and server computers that enables your computer to access html documents that are interconnected through hyperlinks, which form the web.
In order to access these html documents your computer's web browser will need to send in a request to the servers(HTTP request.)
The servers send your computer back an HTML document that your browser can then interpret into what you see as a webpage on your display. This video hosted by Udacity does a good job of explaining more...

HTML Documents

HyperText Markup Language (HTTP) Consist of:

  • Text Content, What you see?
  • Markup, What it looks like
  • References to imgs, vds... hyperlinks
  • Other documents

Tags and Elements

HTML contains elements and tags. Tags around elements or before them defines what type of element and what features our element has.

Computers don't think they just do.

Computers will take instructions given by the user literally and will obey only the instructions it is programmed to inrepret. Therefore, make sure syntax is correct as it protects your project from the computer's limitations.

Inline vs Block Elements

It's easier to group things together than to deal with data as bulk or to try to individualy modify each element in out code. The way we can do this is by making blocks that can create a virtual box and group its containing elements.
Inline elements on the other hand , are within the bulk data and modify small sections of it.
Examples include Block elements such as "div, form, p" and Inline such as "b, br, img, em, strong and span."

Tree-Like Structure of HTML and indenting

Indenting is really important in HTML as indenting can show you which box belongs to what and as you go deeper into the document the more indenting you will see. Make sure to indent in a consistnent style so that you can find it easier to find things within your own code.
There are websites that can indent for your code and help you reach beautiful code such as CleanCSS and FWpolice.

As a result of the indenting we end up with this structure of boxes nested into other boxes and these boxes within boxes and the overall structure will help us form the DOM.

Document object model (DOM):

Cross platform and language independent convention for presenting and attracting objects in HTML and other markup languages. The nodes for every document are organized in a tree-like structure called the DOM tree. Objects in the DOM tree may be addressed and manipulated by using methods on the objects. The public interface of a DOM is specified in its application programming interface (API).
More detailed information on the DOM is availablehere.

In order to accomplish this DOM structure we separate parts of the document to boxes with Use of the div or span tag and other tags.

3:Adding CSS and Style to HTML

Abstraction is Awesome!:

Indeed abstraction is awesome as it helped me avoid the repetition of curtain styling elements in my document by assigning a unified class name for the similar sections and class names for their contents. On the other hand now if I don't like a curtain style in my document it will only take me little editing to change the style.

CSS (cascading style sheets)

CSS stands for Cascading Style Sheets and they give programmers a way to control the style of related HTML elements. This is done by giving similar HTML elements the same class name and then specifying the style that should apply to that class. The different styling functions of CSS can be found listed here.

These various classes allow the user to avoid repition as they can store all the styling data for similar objects and all that is needed is one set of data for the similar elements. CSS is a seperate file that can be refrenced to in the head section of the html document. For more information on CSS documents you will find a good explanantion here on the Mozilla Development Network.

Complete Styled HTML Document

To have an HTML document that has style we will need to either integrate the style into the html document itself or we can do the styling by using two files; The HTML document and a CSS document.
When we have the CSS in a seperate document it can be linked to the html document by using the link tag and specifying that the rel="stylesheet".
